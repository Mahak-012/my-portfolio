## 📖 About

A clean, responsive, and ultra-fast portfolio built with **React** and **Tailwind CSS**.
Designed to showcase my projects, skills, and frontend development journey.

Built with a sleek **dark theme** and **emerald green accents** — focused on speed, clarity, and clean UI.

---

## ✨ Features

- 📱 **Fully Responsive** — desktop, tablet, and mobile
- ⚡ **Ultra-Fast UI** — smooth animations, clean design
- 📂 **Projects Section** — 5+ live projects with GitHub and demo links
- 🛠️ **Skills Section** — interactive progress bars and tech stack tags
- 📬 **Working Contact Form** — integrated with Formspree
- 🌗 **Dark Theme** — professional emerald green aesthetic

## 🚀 Run Locally

```bash
# Clone the repository
git clone https://github.com/Mahak-012/my-portfolio.git

# Go to project directory
cd my-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```
